Entendendo a Bagaça

Abaixo estão descritas alguns exemplos de funcionalidades que normalmente criamos, apesar de as vezes requerer muito html, todo o controle fica na mão do desenvolvedor, podendo manipular com maior precisão aquilo que lhe convier. Para começar a entender e brincar com o SmartClient, primeiros vamos aos conceitos básicos, este plugin se baseia no conceito de que a maior parte do desenvolvimento de qualquer software consiste em 1/3 do tempo trabalhando na interface, logo o plugin visa facilitar este passo, permitindo convencionar algumas caracteristicas básicas do jQuery e ajax, que usamos muitas vezes.

No exemplo abaixo vamos colocar uma caixa de texto e ao digitar algum texto nela será apresentada na linha abaixo uma mensagem no famoso estilo Hello World.

Exemplo
Quem é você?

Olá, Visitante

<input type="text" value="Visitante"

        smart="keyup:{target:'#tmp', dataSource: 'Olá, ' + this.val() }" />

<h4 id="tmp">Olá, Visitante</h4>

Botão que ser acionado busca uma página e coloca seu conteúdo numa div
O texto entrará aqui!

<input value="Buscar Texto" smart="click:{

     source:'Texto-Exemplo,509.raw',

     target:'#buttonsBuscarTexto'

   }" type="button" />

<div id="buttonsBuscarTexto">O texto entrará aqui!</div>

Botão que ser acionado muda seu status = ToggleButton

<input value="Mudar o estado do botão" smart="click:{toggleClass:'source'}" type="button" />
